# 红宝书 (RedBook)

> 具体问题具体分析，在实践中检验真理

## 缘起

红宝书之名，源自毛泽东思想这一中国智慧的结晶。

我们怀着虔诚的态度研读主席著作，试图领会其精神实质。主席著作中蕴含的哲学思想——矛盾分析法、实践论、战略战术原则——与算法解决问题的思路高度契合。本着"古为今用、洋为中用"、"实事求是"的精神，我们尝试用这些思想来指导算法学习。

由于水平所限，红宝书一定存在诸多不足。欢迎提出宝贵意见。

## 核心框架

1. **调查研究**：搞清楚输入、输出、约束
2. **抓住主要矛盾**：暴力为什么不行？瓶颈在哪？
3. **具体问题具体分析**：根据特征选算法
4. **理论与实践结合**：小例子验证，边界处理

## 语录对照

| 主席语录 | 算法应用 |
|---------|---------|
| "没有调查，没有发言权" | 审题、分析输入输出 |
| "研究任何问题，都要从矛盾着手" | 找核心困难 |
| "集中优势兵力，各个击破" | 双指针、二分查找 |
| "牵一发而动全身" | 链表操作 |
| "星星之火，可以燎原" | 动态规划 |
| "敌进我退，敌退我进" | 回溯算法 |
| "人民群众是历史的创造者" | 哈希表 |
| "前途是光明的，道路是曲折的" | 栈与队列 |
